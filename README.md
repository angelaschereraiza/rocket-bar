# Rocket Rock & Metal Bar Bern

Static website for **[Rocket Bar](https://www.rocket-bar.ch)**, a Rock & Metal bar near Bern main station.

Built from scratch using **semantic HTML**, **CSS**, and **vanilla JavaScript**, with a focus on performance, accessibility, responsive design and SEO.

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)

## Features
- Multilingual (DE / FR / EN)
- Responsive layout (desktop & mobile)
- Image carousel with lightbox
- Dynamic drinks & food menus
- SEO & structured data (Schema.org)
